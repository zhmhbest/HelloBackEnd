# [Express](../index.html)

## 环境

```bash
npm init -y
yarn add express
touch index.js
```

`index.js`

```js
const express = require('express')
const app = express()
const port = 3000

app.get('/', (req, res) => {
    res.send('Hello World!')
})

app.listen(port, () => {
    console.log(`Example app listening at http://localhost:${port}`)
})
```
